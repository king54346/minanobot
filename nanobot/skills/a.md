###  `/nanobot/skills/` - 内置技能

| 技能 | 作用 |
|------|------|
| `github/` | GitHub操作（使用 `gh` CLI） |
| `weather/` | 天气查询（wttr.in、Open-Meteo API） |
| `summarize/` | 总结URL、文件、YouTube视频 |
| `tmux/` | 远程控制 tmux 会话 |
| `cron/` | 定时任务管理 |
| `memory/` | 记忆系统使用指南 |
| `skill-creator/` | 创建新技能 |
每个技能包含一个 `SKILL.md` 文件，教会Agent如何使用特定功能。
